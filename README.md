# 📊 DAX Depo (Power BI Project)

## 📌 Overview
This project focuses on advanced DAX functions in Power BI to perform complex calculations and enhance data analysis capabilities. It demonstrates how DAX can be used to create dynamic and insightful dashboards.

## 🚀 Features
- Advanced DAX calculations and measures
- Time intelligence functions (YTD, MTD, YoY)
- KPI creation and analysis
- Dynamic filtering and calculations
- Improved dashboard insights using DAX logic

## 🛠️ Tools Used
- Power BI
- DAX (Data Analysis Expressions)
- Power Query

## 📊 Key Operations Performed
- Creating calculated columns and measures
- Applying time-based calculations
- Building dynamic KPIs
- Using conditional logic in DAX
- Enhancing reports with advanced analytics

## 🎯 Purpose
The goal of this project is to strengthen DAX skills and understand how advanced calculations improve business intelligence reporting.

## 📌 Conclusion
This project highlights expertise in DAX and Power BI, enabling complex data analysis and smarter decision-making.
